Research Project

Localized Voice Banking Assistant for Low Literacy Users

📋 Overview

An AI-powered voice banking assistant designed to help low-literacy and rural users access digital banking services using voice commands in their local language. The system enables users to check balances, transfer money, view transaction history, and make payments—all through speech recognition and voice responses.

🎯 Problem Statement

Traditional banking applications rely on text-based interfaces, creating barriers for elderly users, rural populations, and people with limited literacy. This project eliminates those barriers through voice-based interaction.

🛠️ Technologies Used

  • Speech-to-Text (STT): For accurate voice input processing
  • Natural Language Processing (NLP): For intent recognition and context understanding
  • Text-to-Speech (TTS): For voice responses in local languages
  • JavaScript/React: For responsive web interface
  • Banking API Integration: For secure transactions
  • OTP Authentication: For secure verification

🔄 System Workflow

  1. User logs into banking portal
  2. Voice assistant listens for commands
  3. System processes speech and identifies intent
  4. User confirms transaction details
  5. OTP verification for security
  6. Payment processed and receipt generated
  7. Voice confirmation of successful transaction

💫 Impact

  • Improves financial inclusion for underserved populations
  • Eliminates literacy barriers in digital banking
  • Makes banking accessible to elderly and visually impaired users
  • Published research paper documenting methodology and results
